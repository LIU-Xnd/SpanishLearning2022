## A CON DE PARA y POR

- A
	- 时间
	- 目的地
- CON
	- 方式
	- 伴随
- DE
	- 内容 ¿Pedimos una ensalada de verduras?
	- 原因 Encantado de conocerlos.
	- 整体中的一部分
	- 时间副词短语 Antes de; Después de.
	- 词组 de acuerdo, de nada
- PARA
	- 目的
	- 对象
- POR
	- 原因
	- 地点（动态, 如aquí, allí）¿Qué haces por aquí?
	- 时间
	- 词组 por supuesto, por favor

NOTE: con + mí/tí => contigo/conmigo
